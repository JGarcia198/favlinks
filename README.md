Project 3: FavLinks (React Components, State, and Props)

This project was completed for Project 3 – FavLinks as part of the course assignment.

The goal of this project is to build a React application that allows users to:

View a list of favorite links

Add new links using a form

Remove links dynamically

Practice using React components, props, and state
